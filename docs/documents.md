# Document Generation

[← Main](index.md) · [ruby-dsl](ruby-dsl.md)

## Overview

Nodex generates documents from the same Node tree used for HTML:

```
Node tree → .to_html    (HTML)
          → .to_docx    (DOCX — pure Ruby, zero deps)
          → .to_pdf     (PDF — via DOCX → LibreOffice)
```

The C extension (`nodex-native`) provides additional formats (ODT) and features (images with auto-sizing, headers/footers, page numbers).

## DOCX (Pure Ruby)

```ruby
require 'nodex'
include Nodex::DSL

report = div {
  h1 "Quarterly Report"
  p("Department: Engineering").bold
  
  h2 "Summary"
  p "Revenue increased by 15% compared to Q3."
  
  h2 "Metrics"
  table {
    tr { th "Metric"; th "Q3"; th "Q4" }
    tr { td "Revenue"; td "$1.2M"; td "$1.38M" }
    tr { td "Users"; td "45K"; td "52K" }
  }
  
  h2 "Formula"
  math_block("\\Delta = \\frac{Q4 - Q3}{Q3} \\times 100\\%")

  p "See attached chart."
  img("chart.png", alt: "Chart")
}

# Generate DOCX
report.to_docx("report.docx", preset: :report)
```

### Presets

| Preset | Font | Size | Spacing | Indent | Margins |
|--------|------|------|---------|--------|---------|
| `:gost` | Times New Roman | 14pt | 1.5× | 1.25cm | 30/15/20/20mm |
| `:academic` | Times New Roman | 12pt | 2× | 1.27cm | 25mm all |
| `:report` | Calibri | 11pt | 1.15× | none | 20mm L/R, 25mm T/B |
| `:letter` | Calibri | 11pt | 1× | none | 25mm L/R, 20mm T/B |

Custom options override presets:

```ruby
node.to_docx(preset: :gost, font: "Arial", size: 24)
```

### Supported Elements

| Element | DOCX Output |
|---------|-------------|
| h1—h6 | Heading1—6 styles (bold, sized) |
| p | Paragraph with spacing |
| strong/b, em/i, u, s/del | Bold, italic, underline, strikethrough |
| color, font-size, font-family | Run properties |
| text-align | Paragraph justification |
| ul/ol + li | Numbered/bullet lists (nested) |
| table/tr/th/td | Tables with borders, bold headers |
| a(href) | Hyperlinks |
| img(src) | Embedded images (JPEG/PNG) |
| code/pre | Monospace (Courier New), grey background |
| blockquote | Indented italic paragraph |
| math/math_block | Cambria Math italic text |
| page_break | Page break |
| hr | Bottom border paragraph |

### Image Sizing

```ruby
# HTML attributes (pixels)
img("photo.png", alt: "Photo").set_attr("width", "400").set_attr("height", "300")

# CSS styles (any unit)
img("photo.png", alt: "Photo").width("10cm").height("7cm")
```

Supported units: px, pt, cm, mm, in.

## PDF

PDF is generated by converting DOCX through LibreOffice headless:

```ruby
node.to_pdf("output.pdf", preset: :academic)
```

Requires `soffice` (LibreOffice) in PATH. The conversion is:
1. Generate DOCX in memory
2. Write to temp file
3. `soffice --headless --convert-to pdf`
4. Read PDF, clean up

## Math Formulas

### DSL

```ruby
math("x^2 + y^2 = z^2")            # inline
math_block("E = mc^2")              # display (centered)
```

### HTML Output

Uses KaTeX-compatible delimiters. Add `katex_head()` to document:

```ruby
document("Math Page",
  head: katex_head,
  body: [
    p("Inline: ").append(math("x^2")),
    math_block("\\sum_{i=1}^{n} i = \\frac{n(n+1)}{2}"),
  ]
)
```

### DOCX Output

Math is rendered as Cambria Math italic text. Full OMML (Office Math Markup Language) conversion is planned as a separate task.

### Markdown

```markdown
Inline math: $x^2 + y^2 = z^2$

Display math:
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

Both `$...$` and `$$...$$` are parsed in `Markdown.to_html` and `Markdown.to_node`.

## Markdown → Documents

```ruby
# Markdown → Node tree → DOCX
node = Nodex.markdown_node(File.read("article.md"))
node.to_docx("article.docx", preset: :academic)

# With styling
node = Nodex.markdown_node(md_text) | Padding(20)
node.to_docx("styled.docx", preset: :gost)
```

## C Extension (nodex-native)

The C extension provides additional features:

| Feature | Pure Ruby | C Extension |
|---------|-----------|-------------|
| Headings, paragraphs, lists | ✓ | ✓ |
| Bold, italic, color, font | ✓ | ✓ |
| Tables with borders | ✓ | ✓ |
| Hyperlinks | ✓ | ✓ |
| Images (embedded) | ✓ | ✓ |
| ODT format | — | ✓ |
| Headers/footers | — | ✓ |
| Page numbers | — | ✓ |
| Colspan/rowspan | — | Partial |
| GOST/academic/report presets | ✓ | ✓ |
| Custom TTF fonts | — | ✓ |

Use the C extension when you need ODT or advanced features:

```ruby
require 'nodex/native'
Nodex::Doc.to_docx(node, preset: :gost, header: "Report Title")
Nodex::Doc.to_odt(node, preset: :gost)
```

## CLI

```bash
nodex-ruby -e 'require "nodex"; include Nodex::DSL
  div { h1 "Hello"; p "World" }.to_docx("hello.docx", preset: :report)'
```

Works from any directory — `nodex-ruby` is installed globally via nix profile.
